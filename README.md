# Wired2Scale AI Platform Demo

A working prototype for an AI-powered creator partnership ads operations platform.

## Features

- Internal creator operations dashboard
- Masked client portal view
- Creator sourcing and scoring
- Campaign performance analytics
- Mock Meta / Northbeam data feed
- AI brief and script generation workflows
- Access control and data masking concept
- Reporting dashboard

## Tech Stack

- Next.js
- TypeScript
- Tailwind CSS
- LocalStorage demo data
- Mock AI workflows
- Mock ad-data integrations

## Run Locally

```bash
npm install
npm run dev
```

Open:

```text
http://localhost:3000
```

## Purpose

This demo was created to showcase how a creator-whitelisting / partnership-ads operation can manage creators, client reporting, masked views, automated performance data, and AI-assisted campaign workflows from one platform.
